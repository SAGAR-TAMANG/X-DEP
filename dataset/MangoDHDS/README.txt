
MangoDHDS: Mango Diseased Healthy Dataset
==========================================

Dataset Description:
--------------------------------------------------------------------------------------------------------------------------
This dataset contains mango fruit images categorized into the following five classes:
1. Anthracnose
2. Bacterial Black Spot
3. Scab
4. Stem End Rot 
5. Healthy
The dataset is intended for machine learning applications in fruit disease detection, particularly mango disease classification tasks.

Structure:
--------------------------
images
  ├── An1.jpg to An300.jpg
  ├── Bc1.jpg to Bc300.jpg
  ├── Sc1.jpg to Sc300.jpg
  ├── Se1.jpg to Se300.jpg
  ├── He1.jpg to He300.jpg


Example Entry:
-------------------------
An1.jpg, Anthracnose
Bc1.jpg, Bacterial Canker
Sc1.jpg, Scab
Se1.jpg, Stem End Rot
He1.jpg, Healthy

Usage:
----------
This dataset can be used for:
- Training and evaluating classification models
- Research in agricultural disease detection
- Comparative studies in image-based disease diagnosis
